Привіт! Короткий опис портфоліо
-------------------------------
Папка "Implementation of Basic Algorithms on C++" містить імплементацію різноманітних алгоритмів, наприклад алгоритму пошуку або створення List.
-------------------------------
Папка "Unity Experience" містить assets проектів, реалізовані в рушію Unity. Опис проектів:
- Foxxy Adventure. Реалізація простого платформеру с ворогами, аналог Супер Маріо;
- Minesweeper. Реалізація гри "Сапер";
- Puzzle. Реалізація сцени, де потрібно зібрати паззл;
- Remember Numbers. у всіх ячійок скриті номери, необхідно попарно знайти ячійки з однаковими номерами;
- Sort Numbers. Реалізація п'ятнашок;
- Space War. Простенький аналог гри "Galaxians";
- Tetris. Реалізація тетрісу.
